## NH futures API Trader

*NH선물 API를 사용하여 선물 자동 매매를 구현한 프로젝트입니다.*

*[NH선물 홈페이지 - 거래시스템 - API - 자료실](https://www.futures.co.kr/content/Getcontent.do?content=3000034#tab3)에서 파이썬 샘플 파일 다운로드 후 login.py, main.py가 있는 폴더 내 config구성설정 파일에서 본인 아이디, 비밀번호, 공동인증서 비밀번호 입력&저장 후 코드 사용*

**기술스택** 

- Python 3, anaconda
